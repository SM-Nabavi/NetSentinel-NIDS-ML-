# NetSentinel NIDS-ML
 NetSentinel: AI-Powered Network Intrusion Detection System(NIDS-ML)

## 📋 Project introduction  

**NetSentinels NIDS-ML** is a network intrusion detection system (NIDS) that analyzes network traffic and identifies security threats using machine learning algorithms, as well as in parallel with an IDS.
